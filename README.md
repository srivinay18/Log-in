Over View:
       This project is a Streamlit web application that provides user registration and login functionality. It demonstrates how to build interactive authentication workflows using Python and Streamlit, with options to extend into database integration for persistent storage.

Credentials File Setup:
       This app requires a credentials file to manage user authentication.Created a file named login_credentials.py

Registrtion Email Vaidaition: 
      
  The app includes a registration.py module that handles user registration and validates email/password eligibility.
** Validation Rules**
- Email must contain at least one uppercase letter (A–Z)
- Email must contain at least one lowercase letter (a–z)
- Email must contain at least one number (0–9)
- Email must follow a valid format (e.g., user@example.com)
- Password minimum length should be 6 and maximum lentgh must not be greater than 16
